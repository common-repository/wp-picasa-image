Insert pictures from Picasa album by using the picture address.


== Description ==

You just need to copy the address of the pictures from Picasa album, then this plugin will help you modify the address to show in you posts. If the address of picture is not from Picasa, it won't do anything.
The plugin has been embeded into backstage editor, so you don't need to modify the theme files.

If you found any problem, please <a href="http://pigsky.net/plugin-wp-picasa-image">let me know</a>.

== Installation ==

1. Upload `wp-picasa-image` folder to the `/wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If you want to insert a picture when publishing a post, click the `picasa logo`, input the picture from picasa or other albums and click the `insert into Post` button
